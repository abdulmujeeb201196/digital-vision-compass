# Digital Vision Compass

A modern dashboard application built with React and TypeScript to visualize digital strategy metrics through an interactive compass interface.

## Features
- **Interactive Dashboard**: Visual data representation with dynamic metrics
- **Real-time Analytics**: Track and monitor key performance indicators
- **Responsive Design**: Works across desktop and mobile devices
- **Custom UI Components**: Modular elements for flexible design

## Tech Stack
- **Frontend**: React 18 with TypeScript
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS
- **UI Library**: shadcn/ui components
- **Development**: Hot reloading, ESLint, TypeScript

## Getting Started

### Prerequisites
- Node.js 18.0 or higher
- npm or yarn

### Installation
1. **Clone the repository**
   ```bash
   git clone https://github.com/abdulmujeeb201196/digital-vision-compass.git
   cd digital-vision-compass
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Open your browser**
   Navigate to `http://localhost:5173`
